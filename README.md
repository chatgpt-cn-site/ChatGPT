## 💡 如何高效使用 ChatGPT：最佳提问技巧与实用案例

### 提问技巧
1. **提供上下文**：让 ChatGPT 更好理解问题背景。
   - 示例：**“假设你是一名营销专家，请帮我制定一个推广计划。”**
2. **具体化需求**：说明你的问题需要输出的形式。
   - 示例：**“帮我生成一个表格，列出市场营销的关键指标。”**
3. **分步提问**：将复杂问题拆分成多个小问题。
   - 示例：**“先列出三种推广方法，每种方法的优缺点是什么？”**
