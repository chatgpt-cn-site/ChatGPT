## 数据隐私与安全性说明

利用 ChatGPT 进行智能对话时，请严格注意以下事项：
- **避免泄漏敏感信息**：包括身份证号码、银行账户信息等。
- **优选加密访问**：建议使用 HTTPS 加密连接，确保您的数据安全。
- **谨慎输出结果**：检查 AI 生成内容，避免内容误导或偏差。
